变类型建筑

        有些建筑可以自动或通过玩家的操作改变自身类型。 这些建筑被称为变类型建筑。在获得收益和计算分数时，这些建筑的类型为其当前类型。
        变类型建筑 默认不加入游戏。房主可以决定是否加入这些建筑。若房主决定加入，2张技校，2张奶茶店， 1张路障和1张新闻大厦会加入牌堆。 招待所，办事处， 商务宾馆，和写字楼中会随机2张加入牌堆。

     技校

        你回合结束时该建筑会改变类型，顺序为：
            教育（蓝色） -> 商业（绿色） -> 治安（红色） -> 市政（黄色）-> 教育（蓝色） -> ...
        如此循环。该效果自动发动，你无法避免。
        技校的初始类型为教育（蓝色）。

     奶茶店

        虽然价格和类型相同， 奶茶店和菜场、小吃店不同名。
        加入奶茶店的目的是为了平衡商业建筑的数量。

     路障

        你可以免费建造路障， 建造路障不占用你的建筑回合上限。本回合若你已建造了建筑， 你依然可以建造路障。 虽然免费，路障是一个合法的建筑，所以如果建造路障后达到游戏结束的建筑上限， 当回合结束后游戏结束。
        你下一次建造后拆除路障， 该效果立即自动发动，你无法避免。

     新闻大厦

        你执行行动后从牌堆顶放置1张牌于该牌最下方。回合结束时若该建筑下方有2张牌， 你立即得到该2张牌，该建筑失去特效且变成特殊（紫色）类型。该效果立即自动发动，你无法避免。正常情况下，你建造该建筑后2回合会获得2张牌，同时该建筑改变其类型。
        该建筑改变类型后是本游戏有史以来第一张无效果描述的特殊（紫色）建筑（反倒是其市政类型的形态有）。

     招待所

        你执行行动前展示牌堆顶的一张牌并将其放置于牌堆底，如果展示的牌不为双类型， 招待所的类型变为展示牌的类型。该效果立即自动发动，你无法避免。
        若展示的牌为双类型，该建筑不会改变类型。

     办事处

        你建造完该建筑后可以立即弃置一张牌。 若你以此法弃置的牌为单类型，该建筑的类型变为你弃置牌的类型。在此之后你无法通过弃置牌来改变该建筑的类型。
        建造完该建筑后你可以选择不弃置任何牌， 若如此做，该建筑仍为特殊（紫色）类型。

     商务宾馆

        你之后每建造一个单类型的建筑， 商务宾馆的类型会变为和你新造的建筑相同。该效果立即自动发动，你无法避免。
        若你建造一个双类型建筑，该建筑不会改变类型。

     写字楼

        你可以在回合内展示一张单类型的牌， 并将写字楼的类型变更为你展示的牌的类型。每回合限一次
        若你展示一个双类型建筑，该建筑不会改变类型。
